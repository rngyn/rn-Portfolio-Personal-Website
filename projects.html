yycEats - Course Project

Easily filter through some of my favourite restaurants in this webapp that utilizes Google Maps and Foursquare's APIs. Animated map markers are hidden from view when filtered, and clicking a marker shows a pop up info window. Information from Foursquare includes the name, address, and phone number of the selected restaurant.

Built using HTML, CSS and JavaScript (jQuery and knockout.js)
